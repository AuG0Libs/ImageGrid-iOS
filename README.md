ImageGrid-iOS
=============

Image Grid animation based on single UIImage. Pass in UIImage and number of columns, rows and presto!

[[ImageFlipGrid alloc]initWithImage: [UIImage imageNamed:@"testImage.png"] gridCount:CGSizeMake(15,3)];
